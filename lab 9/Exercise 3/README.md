# Lab 9: Exercise 3 - React Counter Application

## Overview

This project implements a simple, interactive counter system using React functional components and the `useState` hook. It demonstrates dynamic data updates and event handling in a modern React environment.

## Features

- **Functional Component**: Built using React's functional paradigm.
- **State Management**: Uses the `useState` hook to manage and track the counter value.
- **Dynamic UI**: Automatically re-renders the interface whenever the counter value changes.
- **Interactive Controls**: Features increment and decrement buttons with event handling.
- **Modern Design**: A clean, subtle user interface with responsive layout.

## Technologies Used

- React 19
- Vite (Build Tool)
- CSS (Vanilla)

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Navigate to the project directory:

   ```bash
   cd "lab 9/Exercise 3"
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

### Running the Application

To start the development server:

```bash
npm run dev
```

The application will be accessible at the local URL provided by Vite (e.g., `http://localhost:5173`).

## Project Structure

- `src/components/Counter/`: Contains the `Counter.jsx` logic and `Counter.css` styles.
- `src/App.jsx`: Main application container.
- `src/index.css`: Global styles.
