# Lab 9 - Exercise 1: Student Profile (React Basics & JSX)

Welcome to the **Student Profile Application**, developed for **Lab 9 - Exercise 1**. This project introduces the foundational concepts of React, focusing on functional components and dynamic data rendering using JSX.

## 🎯 Objectives

- **Functional Components**: Learning how to structure a UI into reusable functional units.
- **JSX Expressions**: Understanding how to combine HTML-like structure with JavaScript logic.
- **Data Binding**: Using curly braces `{}` to render JavaScript variables directly in the interface.
- **Initial Setup**: Bootstrapping a modern React environment using Vite.

## ✨ Premium Features

- **Modern Aesthetic**: Clean, responsive layout with an elegant "Student Profile" card.
- **Interactivity**: Smooth hover transitions and clear information grouping.
- **Glassmorphism Elements**: Professional UI components with backdrop blur and soft shadowing.
- **Accessibility**: Organized with semantic HTML tags for screen readers.

## 📂 Project Structure

```text
src/
├── components/
│   └── StudentProfile.jsx   # Core functional component
├── App.jsx                  # Main integration component
├── index.css                 # Global styles and layout
└── main.jsx                  # Application entry point
```

## 🛠️ How to Run

Ensure you have **Node.js** and **npm** installed.

1. **Install Project Dependencies**:

   ```bash
   npm install
   ```

2. **Run Development Server**:

   ```bash
   npm run dev
   ```

3. **Access via Browser**:
   Open the local server URL (usually [http://localhost:5173](http://localhost:5173)).
