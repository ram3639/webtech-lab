# Lab 10 Exercise 1: React Form with Validation

## Overview

A high-performance, premium React application designed to collect user information through a secure and validated form interface. This project demonstrates advanced state management with hooks, controlled components, and dynamic UI rendering in React.

## Features

- **Modern Dark UI**: Premium glassmorphism design with smooth animations and HSL-based color theory.
- **Controlled Components**: Synchronous data binding between form inputs and React state.
- **Real-time Validation**: Instant feedback on field requirements and format correctness.
- **Dynamic Error System**: Shake animations and conditional rendering for error states.
- **Secure Handling**: Event-driven submission prevention on invalid inputs.
- **Auto-Reset Logic**: Seamless transition to a fresh form state after successful registration.

## Functional Requirements

- [x] Create a form interface using a **functional component**.
- [x] Manage input fields (name, email, password) using the **useState Hook**.
- [x] Bind input field values using **controlled components**.
- [x] Capture user input changes using the **onChange** event handler.
- [x] Validate input fields (empty check, email regex) using **conditional logic**.
- [x] Display validation messages dynamically using **conditional rendering**.
- [x] Prevent form submission if validation fails using **preventDefault**.
- [x] Handle form submission using the **onSubmit** event.
- [x] **Reset form fields** after successful submission.
- [x] Maintain proper form structure and **dynamic user feedback**.

## Technical Stack

- **Framework**: React 19+
- **Styling**: Vanilla CSS (Custom Variable Design System)
- **Build Tool**: Vite (Lightning-fast HMR)
- **Icons**: Custom SVG paths
